# Hướng dẫn đóng góp

Cảm ơn bạn đã muốn đóng góp cho Otto Robot Việt!

## Quy trình

1. Fork repo này
2. Tạo branch mới: `git checkout -b feature/ten-tinh-nang`
3. Commit với message rõ ràng (tiếng Việt hoặc tiếng Anh đều được)
4. Push và mở Pull Request

## Quy tắc code

- Tuân theo style trong `docs/code_style.md`
- Comment tiếng Việt được khuyến khích cho phần Việt hóa
- Không commit file `sdkconfig`, `build/`, `managed_components/`

## Báo lỗi

Dùng Issues với template có sẵn. Đính kèm:
- Phiên bản firmware (`git describe --tags`)
- Log Serial đầy đủ
- Mô tả phần cứng (board, module, wiring)

## Câu hỏi

Mở Issue với label `question`. Không cần template.
